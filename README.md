  # Claude Code 数学建模实战指南

  用 Claude Code 完成数学建模竞赛论文的完整工作流。

  ## 快速开始

  1. 安装所需 skill（math-modeling、brainstorming、writing-plans 等）
  2. 按 `部署SlashCommand.md` 部署 slash command
  3. 输入 `/mmsetup` 开始

  ## 命令

  | 命令 | 作用 |
  |------|------|
  | `/mmsetup` | 配置语言、格式、比赛类型 |
  | `/mmstep1` | 读取题目和数据 |
  | `/mmstep2` | 方案分析 |
  | `/mmstep3` | 制定计划 |
  | `/mmstep4` | 建模分析 |
  | `/mmstep5` | 编程求解 |
  | `/mmstep6` | 论文撰写 |
  | `/mmstep7` | 打包支撑材料 |

  详见 `ClaudeCode数学建模指南.md`
