# 快速设置（每次做新题目前发一次）

## 提示词

```
本次数学建模的配置信息：
- 编程语言：[Python/MATLAB/其他]
- 论文格式：[Word/LaTeX]
- 比赛类型：[国赛/美赛/其他]
- TDD 范围：[不需要 / 对核心算法做TDD]

请记住以上配置，后续步骤中直接使用，不要再问我。
```

## 说明

- 这段话在开始 `/step1` 之前发一次即可
- Claude 会记住这些配置，后续步骤中不再重复询问
- 如果中途想改配置，重新发一次就行
- 把 `[Python/MATLAB/其他]`、`[Word/LaTeX]` 等替换为你的实际选择

## 示例

```
本次数学建模的配置信息：
- 编程语言：Python
- 论文格式：Word
- 比赛类型：国赛
- TDD 范围：不需要

请记住以上配置，后续步骤中直接使用，不要再问我。
```
